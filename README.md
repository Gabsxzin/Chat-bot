# 🤖 Chat Bot Portfolio

Um chatbot desenvolvido em **Python** com interface gráfica utilizando **CustomTkinter**, criado para apresentar meu perfil profissional de forma interativa.

<img width="1624" height="650" alt="image" src="https://github.com/user-attachments/assets/35d2c2c8-26e9-4aad-8c8d-2c43ca941188" />


## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de criar um portfólio diferente dos modelos tradicionais. Através de uma interface simples e intuitiva, o usuário pode conhecer mais sobre minha trajetória acadêmica, habilidades técnicas, acessar meu currículo e visitar meu perfil no GitHub.

O chatbot funciona através de opções numéricas que retornam informações específicas sobre meu perfil profissional.

---

## 🚀 Funcionalidades

* 👤 Apresentação pessoal
* 📄 Download automático do currículo
* 💻 Exibição das habilidades técnicas
* 🌐 Acesso direto ao perfil do GitHub
* 🖥️ Interface gráfica moderna utilizando CustomTkinter

---

## 🛠️ Tecnologias Utilizadas

* Python 3
* Tkinter
* CustomTkinter
* Selenium
* Wget

## ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/Gabsxzin/Chat-bot.git
```

Entre na pasta do projeto:

```bash
cd Chat-bot
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

---

## ▶️ Executando o Projeto

Execute o arquivo principal:

```bash
python chat.py
```

---

## 📋 Opções Disponíveis

Ao iniciar o programa, serão exibidas as seguintes opções:

| Opção | Descrição             |
| ----- | --------------------- |
| 1     | Informações sobre mim |
